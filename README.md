# Mobile Banking Application (Case Study)

## Overview
A cross-platform mobile banking application developed for a microfinance institution, enabling users perform secure financial transactions and manage their accounts.

This project focuses on delivering reliable, user-friendly financial workflows while maintaining security and performance standards required in banking systems.

---

## Key Features
- Secure authentication and account management
- Internal fund transfers
- External transfers
- Bill payment workflows
- Modular, feature-based architecture
- Cross-platform support (Android & iOS)

---

## Tech Stack
- **Frontend / Mobile:** .NET MAUI, C#
- **Backend:** ASP.NET APIs
- **Database:** SQL Server
